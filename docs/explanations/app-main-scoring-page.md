# app/(main)/scoring/page.tsx

## このファイルは何をするのか
アイデアノートをAIが4軸でスコアリングし、レーダーチャートで結果を可視化するページ。

## なぜこのファイルが必要なのか
スコアリング機能のフロントエンドUI。ノート選択から分析実行、結果の視覚的表示までをワンストップで提供する。

## コードの概要
- `RadarChart` コンポーネント: SVGで4軸のレーダーチャートを描画。5段階のグリッド線、データポリゴン、各軸のラベルとスコアを表示
- アイデアタグのノートを `/api/notes?tag=アイデア` で取得し、セレクトボックスに表示
- スコアリング実行後、レーダーチャート・スコア詳細（ドットインジケーター）・AIコメント・総合スコア（4軸平均）を表示
- 4軸: 実現可能性（青）、インパクト（黄）、低労力（緑）、独自性（紫）

## 依存関係
- 使っているもの: `react`, `next/navigation`, `@/components/ui/button`, `lucide-react`
- 使われているもの: サイドバーナビゲーション
