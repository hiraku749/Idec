# 初回実装 — 要求内容

---

## 概要

Idec の MVP（Minimum Viable Product）を実装する。
「アプリの核心が体験できる最小構成」に絞り、以下の4機能を実装対象とする。

| 対象機能 | 理由 |
|----------|------|
| ノート機能（基本） | アプリの核心・全サービスの基盤 |
| OwnAIエージェント（DB検索のみ） | 差別化機能の体験が可能 |
| 認証 | アプリ成立の必須要件 |
| ダッシュボード（簡易） | UXの入口 |

**MVP対象外（後続フェーズで実装）：**
AI壁打ち / コンテキストエンジニアリング / 文章増強 / ロードマップ生成 / 図式化 / ディスカッション / チーム招待 / 共有機能 / テンプレート / カバー画像・アイコン / カラーラベル / エクスポート

---

## ユーザーストーリー

### 認証

```
As a 新規ユーザー,
I want to メールまたはGoogleアカウントで登録・ログインしたい,
So that すぐにアプリを使い始められる.
```

### ノート機能

```
As a ユーザー,
I want to アイデアやメモをノートとして作成・編集・削除したい,
So that 頭の中の情報を忘れる前に外部化できる.

As a ユーザー,
I want to ノートにタグ（アイデア/情報/ToDo）とユーザータグを付けたい,
So that 後から目的別にすばやく検索・フィルターできる.

As a ユーザー,
I want to ToDoタグのノートにチェックボックスとリマインダーを追加したい,
So that タスクの進捗を管理し、忘れずに実行できる.

As a ユーザー,
I want to ノートを全文検索・タグフィルターで探したい,
So that 過去の情報にすばやくアクセスできる.

As a ユーザー,
I want to ノートをピン留め・アーカイブ・ゴミ箱に移動したい,
So that ノートを整理し、重要なものにすぐアクセスできる.
```

### OwnAIエージェント

```
As a ユーザー,
I want to 自分のノートをナレッジとしてAIに質問したい,
So that 過去に書いた情報を活用した回答を得られる.

As a ユーザー,
I want to AIの回答をノートとして保存したい,
So that 有用な情報をワークスペースに蓄積できる.
```

### ダッシュボード

```

As a ユーザー,
I want to ログイン後に最近のノート・未完了ToDo・ピン留めノートを一覧で見たい,
So that すぐに作業に取りかかれる.
```

## 受け入れ条件

### 認証

- [ ] メールアドレス・パスワードで新規登録できる
- [ ] メールアドレス・パスワードでログインできる
- [ ] Googleアカウントでログインできる
- [ ] 未ログイン状態でアプリにアクセスするとログインページにリダイレクトされる
- [ ] ログアウトするとセッションが切れ、ログインページに戻る

### ノート機能

- [ ] タイトル・本文・タグ（アイデア/情報/ToDo）・ユーザータグを設定してノートを作成できる
- [ ] 作成済みのノートを編集・保存できる
- [ ] ノートを削除するとゴミ箱に移動する（30日後に完全削除）
- [ ] ゴミ箱のノートを復元できる
- [ ] ToDoタグのノートにチェックボックスを追加でき、ON/OFFできる
- [ ] リマインダーに日時を設定できる
- [ ] ノートをピン留めできる
- [ ] ノートをアーカイブできる
- [ ] タイトル・本文で全文検索できる
- [ ] タグ・ユーザータグでフィルター検索できる
- [ ] ノート保存時にベクトル埋め込みが自動生成される

### OwnAIエージェント

- [ ] プロンプトを送信すると、DBのノートをベクトル検索してナレッジとして回答が返る
- [ ] 回答はストリーミング形式でリアルタイム表示される
- [ ] 「ノートに保存」ボタンで回答を新規ノートとして保存できる
- [ ] ナレッジ指定UIでノートを選択してから質問できる
- [ ] free プランのAI使用回数（月30回）を超えると制限メッセージが表示される

### ダッシュボード

- [ ] 最近使ったノートが表示される
- [ ] ToDo未完了一覧が表示される
- [ ] ピン留めノートが上部に固定表示される

### 全体

- [ ] ダーク/ライトモードの切り替えが動作する
- [ ] レスポンシブ対応（スマホ・タブレット）
- [ ] Vercelにデプロイされ、公開URLでアクセスできる

---

## 制約事項

- Supabase Free プラン内（DB 500MB / Storage 1GB）で運用する
- Vercel Hobby プラン内で運用する
- AI応答は Server-Sent Events（SSE）でストリーミングし、Vercel の10秒タイムアウトを回避する
- Web検索APIはMVPでは未実装。OwnAIはDB検索（pgvector）のみ
- 共有・チーム機能はMVPでは未実装
- テンプレート・カバー画像・カラーラベル・エクスポートはMVPでは未実装
