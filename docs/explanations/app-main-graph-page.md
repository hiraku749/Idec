# app/(main)/graph/page.tsx

## このファイルは何をするのか
ノートとプロジェクトの関係をネットワークグラフ（Canvas描画）で可視化するページ。

## なぜこのファイルが必要なのか
ノート間のリンクやプロジェクトとの関係性を視覚的に把握するため。テキスト一覧では見えない知識の構造をネットワーク図として表示し、ノードクリックで該当ページに遷移できる。

## コードの概要
- `/api/graph` からノード・エッジデータを取得
- 簡易フォースレイアウト: 100回のイテレーションで反発力（ノード間）と引力（エッジ接続）を計算し、ノード位置を決定
- Canvas 2D APIでグラフを描画: エッジ（線）→ ノード（円）→ ラベル（テキスト）の順で描画
- 凡例: アイデア（黄）、情報（青）、ToDo（緑）、プロジェクト（ピンク）
- `handleCanvasClick` でクリック位置のノードを検出し、ノートまたはプロジェクト詳細ページへ遷移
- `devicePixelRatio` 対応で高解像度ディスプレイでもシャープに表示

## 依存関係
- 使っているもの: `react`, `next/navigation`, `@/components/ui/button`, `lucide-react`
- 使われているもの: サイドバーナビゲーション
